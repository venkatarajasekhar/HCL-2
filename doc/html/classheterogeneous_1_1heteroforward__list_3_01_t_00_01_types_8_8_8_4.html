void heterogeneous::heteroforward_list< T, Types...>::unique ( BinaryPredicate  binary_pred)
inline

Takes as argument a specific comparison function that determine the "uniqueness" of an element.

Calls binary_pred(i,(i-1)) for all pairs of elements and removes i from Nth container of type U if predicate returns true;
